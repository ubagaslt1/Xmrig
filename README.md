## Installation

1. Add this app from the Umbrel Community Store
2. Open the app and configure your Monero wallet address
3. Set your preferred pool and number of CPU threads
4. Click "Start Mining"

## Configuration

### Required:
- **Wallet Address**: Your Monero wallet (starts with '4')
- **Pool**: Mining pool (default: SupportXMR)
- **Threads**: Number of CPU cores to use

### Recommended Settings:
- Use 50-75% of available CPU cores
- Set CPU usage limit to 75-85%
- Keep donation level at 1% (supports XMRig development)
